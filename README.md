# Josh Matos - Portfolio Website

## Neon Tokyo Synthwave Portfolio

### Files Included:
- `index.html` - Portfolio HTML structure
- `styles.css` - All styling (separate file for easy editing)
- `dark-blue-background-designify.PNG` - Professional headshot

### To Run Locally:
1. Open `index.html` in any web browser
2. Or use VS Code Live Server extension
3. Make sure all 3 files stay in the same folder!

### To Deploy:
1. Upload all 3 files to your web host
2. Keep them in the same directory
3. The HTML file can be renamed if needed

### Tech Stack:
- Pure HTML/CSS (no dependencies!)
- Google Fonts (Orbitron, Rajdhani)
- Responsive design
- Synthwave/Cyberpunk aesthetic

### Customization:

**Quick Color Changes** (in `styles.css`):
Lines 17-22 have CSS variables:
```css
--void: #0a0a14;        /* Background color */
--neon-cyan: #00ffff;   /* Primary accent */
--neon-pink: #ff0080;   /* Secondary accent */
--neon-purple: #b026ff; /* Tertiary accent */
--electric-blue: #0080ff;
```

**Edit Content** (in `index.html`):
- Your name/title: Search for "JOSH MATOS"
- About section: Look for `<section class="section" id="about">`
- Work history: Look for timeline items
- Projects: Search for "Featured Work"
